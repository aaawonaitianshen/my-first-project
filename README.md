# 🎮 我的小游戏合集

这是一个用 HTML + CSS + JavaScript 实现的网页小游戏合集，所有游戏都可以直接在浏览器里玩，也部署在了 GitHub Pages 上。

---

## 游戏列表
### 🐍 贪吃蛇小游戏
- 玩法：方向键控制蛇移动，吃到食物会得分并变长，撞到墙或自己身体游戏结束
- 在线体验：[点击这里玩贪吃蛇](https://aaawonaitianshen.github.io/small-games/snake/index.html)

### 🧱 俄罗斯方块
- 玩法：方向键控制方块移动和旋转，填满一行即可消除得分，堆到顶部游戏结束
- 在线体验：[点击这里玩俄罗斯方块](https://aaawonaitianshen.github.io/small-games/tetris/index.html)
